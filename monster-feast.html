Monster Feast

Skills Used

  • Unity Programming
  • Event Handling
  • Character Creation
Download

Sept 17 - 24, 2021

This was a game I made for MS Paint Jam 2021, I had a little under a week to make it since I was busy that week. This was my first solo project in Unity, and I learned so much about the platform! It's a reverse claw machine game where you drop candies on monsters to watch them grow.

Art Assets

Process

Programming

As I mentioned, this was my first time programming in Unity, so there was a LOT to learn while making Monster Feast. I learned about how Unity handles animations, calling events, and 2d collision handling.

Character Creation

All art assets had to be made in MS Paint, so it was a struggle initially getting used to not having my texture brushes or ipad. For that reason, I decided to keep the art style simple and not worry too much about shading. Instead, I chose to focus on making monsters that were distinct and reflected their unique traits.
